# Library Manager

Sistema de gerenciamento de biblioteca desenvolvido em ASP.NET Core.

## Funcionalidades

- Gerenciamento de livros
- Gerenciamento de usuários
- Sistema de empréstimos
- Recuperação de senha

## Tecnologias

- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- CQRS com MediatR
- Mailgun para envio de e-mails

## Como executar

1. Clone o repositório
2. Configure as variáveis de ambiente ou user secrets
3. Execute `dotnet restore`
4. Execute `dotnet run`
